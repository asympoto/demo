🔧 EASY FRONTEND INTEGRATION
═════════════════════════════════════════════════════════════════════════════

Just include api-helper.js in your HTML:

    <script src="api-helper.js"></script>

Then use in JavaScript:

    // Login
    await api.login('user@example.com', 'password');

    // Create Order
    await api.createOrder(email, cart, total, address, phone);

    // Get Orders
    const orders = await api.getUserOrders(email);

All pre-built methods with error handling included!

🐛 TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

Issue: "npm install failed"
Solution: Make sure Node.js is installed (nodejs.org)
         Check C:\ drive has at least 500MB free space

Issue: "Port 3000 already in use"
Solution: Change PORT in server.js to 3001
         OR kill existing process on port 3000

Issue: CORS errors in browser
Solution: Check backend is running on http://localhost:3000
         Check api-helper.js has correct baseURL

Issue: Data files not created
Solution: Backend creates them automatically
         Check data/ folder permissions
         Ensure hardware/ folder is writable

Issue: "Cannot find module"
Solution: Run 'npm install' again
         Delete node_modules and reinstall

See BACKEND_SETUP.md for more solutions!
